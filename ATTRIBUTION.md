# Data Attribution

## London Underground Network Data

This application uses Transport for London (TfL) network data compiled and made available by Oliver O'Brien.

### Source Files

- **TfL Lines**: https://github.com/oobrien/vis/blob/master/tubecreature/data/tfl_lines.json
- **TfL Stations**: https://github.com/oobrien/vis/blob/master/tubecreature/data/tfl_stations.json

### Transport for London

The data represents the Transport for London network. TfL also provides official open data:
- [TfL Open Data Users](https://tfl.gov.uk/info-for/open-data-users/)
- [TfL Unified API](https://api.tfl.gov.uk/)

Line colors are based on official TfL branding guidelines.

## Usage

This is a non-commercial educational/demonstration project. The TfL network data is used under the CC-By-NC license for non-commercial purposes only.

For commercial use, please:
1. Obtain appropriate licensing from TfL
2. Comply with OpenStreetMap ODbL requirements
3. Contact Oliver O'Brien regarding the compiled dataset
